# Desafio API QA

Projeto de testes automatizados da API pública [Serverest.dev](https://serverest.dev)

## Ferramentas
- Java 17
- Maven
- JUnit 5
- RestAssured
- JSON Schema Validator

## Funcionalidades testadas
### Produtos
1. Listar produtos existentes
2. Criar novo produto com sucesso

### Carrinhos
1. Listar carrinhos existentes
2. Criar carrinho com produto válido

## Execução
```bash
mvn test
```
